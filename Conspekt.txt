 dispaly: flex - делает все элементы флексами
 justify-content - выравнивание по оси X
    flex-start - содержимое выравнено по левому краю (по умолчанию)
    flex-end -
 
 
align-items - выравнивание по поперечной оси Y (компоненты выравнивает)
    baseline - выравнивание по базовой линии
    stretch - растягивает по всей высоте контейнира (по умолчанию)
    
align-content - выравниввние строк по свободному пространству флекс контейнера

align-self - выранивание самого компонента (игнорируя align-items)



    
flex-wrap - отвечает за перенос содержимого
    nowrap (по умолчанию)
    wrap -разрешает перенос содержимого на новые строки
    wrap-reverse - меняет старт
    
flex-direction: - меняет направление оси
    column - колонка
    row - строка


    Виды элементов:
блочный
строчный 
блочно-строчный
